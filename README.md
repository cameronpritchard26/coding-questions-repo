# coding-questions-repo
A repo for holding generated solutions to similar coding questions I've seen on tests

## Problems

### Additive Numbers

**File:** `additive_numbers.py`

Determine if a number's digits can be partitioned to form an additive sequence where each number equals the sum of the previous two.

See [README_ADDITIVE_NUMBERS.md](README_ADDITIVE_NUMBERS.md) for detailed documentation

### Connect 4 Scoring

**File:** `connect4.py`

Calculate points for a player on a Connect 4 board based on sequences of their pieces (horizontal, vertical, diagonal).

See [README_CONNECT4.md](README_CONNECT4.md) for detailed documentation
