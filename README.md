# coding-questions-repo
A repo for holding generated solutions to similar coding questions I've seen on tests

## Problems

### Additive Numbers

**File:** `additive_numbers.py`

Determine if a number's digits can be partitioned to form an additive sequence where each number equals the sum of the previous two.

See [README_ADDITIVE_NUMBERS.md](additive_numbers/README_ADDITIVE_NUMBERS.md) for detailed documentation

### Connect 4 Scoring

**File:** `connect4.py`

Calculate points for a player on a Connect 4 board based on sequences of their pieces (horizontal, vertical, diagonal).

See [README_CONNECT4.md](connect4/README_CONNECT4.md) for detailed documentation

### Breaking Records

**File:** `breaking_records.py`

Count the number of times a player breaks their season record for most and least points scored in a game.

See [README_BREAKING_RECORDS.md](breaking_records/README_BREAKING_RECORDS.md) for detailed documentation
